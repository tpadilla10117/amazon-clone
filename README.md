# Project Description:

A replica of Amazon using Next.js, Tailwind CSS, Redux, Firebase, and NextAuth

# Installation Steps
`npm install @heroicons/react` -> for hero icons


## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```
